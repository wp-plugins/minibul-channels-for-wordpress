== Description ==

Minibul Channels for WordPress displays a Widget that shows the latest message from any public Minibul Channel in any Minibul Community. If you are looking for a WordPress plugin to show updates from a Minibul, use the minibul-for-wordpress plugin.

**Features**

    *  Simply
    *  Customizable
    *  Widget support
    *  No options page (yes, its a feature)
    *  Uses Wordpress resources (no extra files needed)

**Usage**

If you use WordPress widgets, just drag the widget into your sidebar and configure.
Be sure to have activated the plugin in your Plugin Settings.

== Installation ==

Drop the minibul-for-wordpress folder (or only the .php file) into /wp-content/plugins/ and activate the plug in the Wordpress admin area. Get the Minibul ID (this is a number which you can find on the Follow & Share page) en install it.
